# sEMG 手环 — 硬件说明文档

> 版本：v2.0.0 | 更新：2026-06-27 | 云方案版本（已移除 SPI Flash）

---

## 1. 系统概览

本项目为基于**表面肌电（sEMG）**的肌肉疲劳监测手环系统，由以下三部分组成：

```
人体肌肉
   ↓ 肌电信号（±1.5mV 差分）
干电极传感器模块（思知瑞，增益 ~1000×，输出 0~3.0V）
   ↓ 模拟电压，接 A0
Arduino UNO R4 WiFi（RA4M1 Cortex-M4F, 48MHz）
   ├── 14-bit ADC 采样（5V ref, 0.305mV/LSB）
   └── WiFi 上传云端
   ↓ WiFi（HTTP / 云函数）
微信小程序 + 云端（4 页面：配网 / 校准 / 实时监测 / 数据分析）
```

**云方案说明：**
- 实时数据通过 WiFi 直接上传云端，无本地大容量存储
- 校准参数和用户信息存储于板载 EEPROM（8KB Data Flash）
- 已移除外挂 SPI Flash（W25Q128）及电平转换电路，简化硬件

## 2. 核心器件

### 2.1 主控：Arduino UNO R4 WiFi

| 参数 | 值 |
|------|-----|
| MCU | Renesas RA4M1（ARM Cortex-M4F）@ 48 MHz |
| SRAM | 32 KB |
| 程序 Flash | 256 KB |
| Data Flash | 8 KB（EEPROM 模拟，存 WiFi 凭证 + 校准数据） |
| ADC | 14-bit SAR，0~5V 参考电压，16,383 满量程 |
| 板载 WiFi 模块 | ESP32-S3-MINI，通过 UART(Serial1) 与 RA4M1 通信 |
| 编译框架 | PlatformIO（renesas-ra 平台，arduino 框架） |

### 2.2 肌电传感器模块

| 参数 | 值 |
|------|-----|
| 类型 | 思知瑞干电极 sEMG 模块 |
| 检测范围 | ±1.5 mV（差分肌电信号） |
| 增益 | ~1000~1500× |
| 输出范围 | 0~3.0V（中心偏置约 1.5V） |
| 有效频带 | 10~500 Hz |
| 供电 | 5V / ≥20mA |
| 接口 | XH2.54-3P：红(5V)、黑(GND)、黄(AOUT→A0) |
| 电极 | PJ-342 干电极，50cm 导联线 |

**关键信号量级：**

| 状态 | 传感器输出电压 | 对应 ADC 读数（14-bit 5V ref） |
|------|----------------|-------------------------------|
| 放松/静息 | 1.5V ± 0.1V | ~4915 ± 328 |
| 轻度收缩 | 1.5V ± 0.3V | ~4915 ± 983 |
| 最大随意收缩 (MVC) | 1.5V ± 1.0V | ~4915 ± 3277 |

## 3. 电路连接

### 3.1 传感器 → Arduino

```
sEMG 模块               Arduino UNO R4 WiFi
  VCC（红）    ──────────  5V
  GND（黑）    ──────────  GND
  AOUT（黄）   ──────────  A0（模拟输入）
```

**连接说明：**
- 传感器输出最高 3.0V，UNO R4 WiFi 的 ADC 量程为 0~3.3V（5V 供电时 0~5V），无需电平转换，直接连接即可
- 红线和黑线不可接反，否则可能损坏传感器

### 3.2 板载外设

| 引脚 | 功能 |
|------|------|
| LED_BUILTIN | 板载 LED，低电平点亮（启动阶段指示） |
| D2 | 蜂鸣器（预留） |
| D3/D4/D5 | RGB LED（预留，B/G/R） |

## 4. 硬件安装与调试

### 4.1 传感器安装位置

- 电极应沿肌纤维方向贴附（通常与肌纤维平行）
- 参考电极（GND）贴在电极之间或肌肉末端
- 皮肤干燥时可用酒精擦拭，提高信号质量

### 4.2 信号质量判断

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| RMS 异常低（< 0.01mV） | 电极脱落或接触不良 | 重新贴附电极 |
| MDF 超出范围（< 10Hz 或 > 250Hz） | 噪声干扰或电极位置错误 | 检查电极位置，远离电源线 |
| Quality < 50 | 信号质量差 | 调整电极位置，保持皮肤干燥 |

### 4.3 电源要求

- 供电：USB 5V（通过 USB-C 接口）
- 电流：≥ 500mA（WiFi 连接时峰值电流约 300mA）
- 推荐使用电脑 USB 口或 5V/2A 电源适配器

---

## 5. 已知硬件限制与注意事项

1. **传感器贴附** — 干电极需与肌纤维方向一致；皮肤干燥或接触不良导致信号不稳定。
2. **WiFi 距离** — 设备 WiFi 信号较弱，测试距离建议 1 米内。
3. **ADC 参考** — 使用板载 5V 参考，可改用 3.3V ref 提高 sEMG 信号分辨率。
4. **BLE 广播** — WiFi 连接后广播会暂停，需要重新配网时需重启设备。
5. **电极寿命** — 干电极使用寿命约 20~30 次测量，电极污染后需更换。
