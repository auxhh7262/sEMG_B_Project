# 1. 项目背景与意义

肌肉疲劳的实时监测在运动科学、康复医学和人机交互中具有重要意义。表面肌电信号（sEMG）是一种通过皮肤表面电极采集肌肉电活动的非侵入式技术，已广泛应用于肌肉状态评估。

传统 sEMG 疲劳检测设备通常依赖昂贵的专用仪器和计算机端软件，不便携且成本高。本项目基于 Arduino UNO R4 WiFi 开发了一款便携式 sEMG 疲劳监测设备，具有以下特点：

- **低成本**：BOM 成本约 200 元人民币（sEMG 单通道干电极肌电传感器 + UNO R4 WiFi）
- **便携**：腕带式设计，可实时在手机微信小程序上查看数据
- **云端**：数据通过 WiFi 上传至微信云开发平台，支持历史分析
- **自主算法**：固件端实现完整的 RMS、MDF、激活度、疲劳度计算，无需计算机端处理

本算法说明文档面向高中生科创项目评审，重点阐述核心算法的原理、创新亮点及优缺点。

# 2. 算法整体架构

算法运行在 Arduino UNO R4 WiFi（瑞萨 RA4M1 芯片，主频 48MHz，RAM 32KB）上，采用定时器中断驱动采样，在主循环中完成全部信号处理。

**处理流程（8 步）**

| 步骤 | 处理内容 | 输出 |
|------|---------|------|
| 1 | ADC 中断采样（1000Hz） | 原始 sEMG 电压信号 |
| 2 | 环形缓冲区管理（512 点） | 滑动数据窗口 |
| 3 | RMS 计算（256 点窗口） | 肌肉激活强度（mV） |
| 4 | FFT + 功率谱（256 点） | 频谱分布 |
| 5 | MDF 计算（中值频率） | 频谱中心位置（Hz） |
| 6 | 激活度归一化 | 0～100% |
| 7 | 疲劳度计算（MDF 下降率） | 0～100% |
| 8 | 信号质量评估 | 0～100 分 |

> **关键设计决策**：由于 RA4M1 的 RAM 仅 32KB，FFT 使用 256 点（而非 512 点），以减少内存占用。频谱计算采用自实现定点 FFT（库仑复数旋转因子预计算），避免依赖大型 FFT 库。

# 3. 核心算法详解

## 3.1 信号采集与预处理

sEMG 干电极肌电传感器输出 0~3.0V 模拟信号，直接接入 UNO R4 WiFi 的 14 位 ADC（量程 0～3.3V）。采样由硬件定时器中断驱动，固定 **1000Hz**：

```cpp
// main.cpp — ADC 定时器配置
ADC::adc_timer.begin([]() {
  int16_t adcVal = analogRead(ADC_PIN);
  gSignalProcessor.isrPushSample(adcVal);
}, 1000.0f);  // 1000 Hz
```

**环形缓冲区设计**：使用 512 点的环形缓冲区（`RING_BUFFER_SIZE = 512`），写指针由中断上下文更新，读指针由主循环更新。采用 `volatile` 变量保护，避免中断与主循环之间的数据竞争。

**DC 偏移去除**：sEMG 信号含有电极与皮肤接触产生的 DC 偏移，计算 RMS 前需去除。采用窗口内简单均值法（而非去极值均值），因为 256 点已足够稳定，且去极值会破坏时序：

$$
DC_{bias} = \frac{1}{N} \sum_{i=0}^{N-1} x[i]
$$

其中 $N = 256$（FFT 窗口大小），$x[i]$ 为 ADC 原始采样值（单位：mV）。

## 3.2 RMS（均方根）计算

RMS 反映肌肉收缩时的信号总功率，与收缩强度正相关，是激活度归一化的基础。

$$
RMS = \sqrt{\frac{1}{N} \sum_{i=0}^{N-1} (x[i] - DC_{bias})^2}
$$

- $N = 256$，对应 256 ms 时间窗口（@1000Hz 采样）
- $x[i]$ 为去除 DC 偏移后的交流分量（单位：mV）
- RMS 计算使用加窗**前**的原始信号，因为 RMS 反映时域能量，加窗会影响幅值准确性

## 3.3 MDF（中值频率）计算

MDF（Median Frequency，中值频率）是肌电信号频谱的中心位置，肌肉疲劳时 MDF 会系统性下降。这是本算法的核心疲劳指标，大量文献已验证其可靠性。

**FFT 实现**：使用 256 点实数 FFT（自实现，旋转因子预计算），加 Hamming 窗减少频谱泄露：

```
w[i] = 0.5 × (1 - cos(2πi / (N-1)))  // Hamming 窗
x_windowed[i] = (x[i] - DC_bias) × w[i]
```

**MDF 定义**：MDF 是使下式成立的频率 $f_{mdf}$：

$$
\int_{f_{min}}^{f_{mdf}} P(f) df = \int_{f_{mdf}}^{f_{max}} P(f) df = \frac{1}{2} \int_{f_{min}}^{f_{max}} P(f) df
$$

- $P(f)$ 为功率谱密度
- $f_{min} = 10\ Hz$，$f_{max} = 250\ Hz$（肌电信号有效频谱范围）

**MDF 计算步骤**：

1. 计算功率谱：$P[k] = (Re[k]^2 + Im[k]^2) / N$，$k = 0 \sim N/2-1$
2. 计算总功率：$P_{total} = \sum P[k]$（k 对应 10～250 Hz）
3. 若 $P_{total} < 10^{-12}$，视为信号太弱，返回 -1（跳过本次）
4. 累积功率，找到使累积功率 $\ge 0.5 \times P_{total}$ 的 bin
5. 线性插值获得精确 MDF 值

**EMA 平滑**：原始 MDF 值逐帧波动较大，采用指数移动平均（EMA）平滑：

$$
MDF_{current} = MDF_{last} \\times (1 - \\alpha) + MDF_{raw} \\times \\alpha
$$

| 信号状态 | $\alpha$ 值 | 设计理由 |
|---------|----------|---------|
| 收缩期 / MDF 下降 | 0.35 | 快速跟踪频谱变化，避免滞后 |
| 稳态期（>10 帧） | 0.15 | 抑制逐帧抖动，输出平滑 |
| 启动过渡（<10 帧） | 0.5→0.15 线性过渡 | 避免初始值偏差导致长时间收敛 |

## 3.4 激活度（Activation）计算

激活度将实时 RMS 归一化到 0～100%，直观反映肌肉收缩强度。

$$
A(\\%) = \\text{clamp}\\left( \\frac{RMS_{current} - RMS_{relax}}{RMS_{active} - RMS_{relax}} \\times 100\\%,  0, 100 \\right)
$$

- $RMS_{relax}$：校准得到的放松状态 RMS（10 秒静息均值）
- $RMS_{active}$：校准得到的主动收缩 RMS（15 秒最大收缩峰值）
- **物理意义**：$A = 0\%$ 表示完全放松，$A = 100\%$ 表示达到了校准时的最大收缩强度

## 3.5 疲劳度（Fatigue）计算

疲劳度是本算法的核心输出，基于 MDF 下降幅度计算，归一化到 0～100%。

**公式**：

$$
F_{raw}(\\%) = \\text{clamp}\\left( \\frac{MDF_{baseline} - MDF_{current}}{MDF_{baseline}} \\times 100\\%,  0, 100 \\right)
$$

- $MDF_{baseline}$：**动态基线**，取本次收缩开始时的 MDF 值（A 从 <20% 跨越到 >20% 时捕获）
- 若无可用的动态基线，则使用校准得到的 $MDF_{relax}$

**EMA 平滑**：疲劳度变化是缓慢的生理过程（10～60 秒时间尺度），使用 $\alpha = 0.1$ 的 EMA 平滑：

$$
F_{smoothed}(t) = F_{smoothed}(t-1) \\times 0.9 + F_{raw}(t) \\times 0.1
$$

> **创新点：动态基线设计**。传统方法使用校准时的 $MDF_{relax}$ 作为固定基线，存在日间漂移问题（电极位置变化、皮肤阻抗变化等）。本算法采用「每次收缩独立基线」：每次肌肉收缩开始时自动捕获当前 MDF 作为基线，这样即使电极有轻微位移，也能正确反映本次收缩内的疲劳趋势。

## 3.6 信号质量评估

信号质量（0～100 分）实时反映当前 sEMG 信号是否可信，用于用户界面提示用户调整电极位置。

$$
Quality = RMS\\_score + MDF\\_score + Continuity\\_score = 35 + 35 + 30 = 100
$$

**评分规则**：

- **RMS 得分（35 分）**：收缩期 RMS 在 0.1～5 mV 为高质量；放松期 RMS < 0.5 mV 为高质量
- **MDF 有效性（35 分）**：$MDF \in [10, 250]\ Hz$ 范围内得满分
- **连续性（30 分）**：过去 50 帧中 MDF 有效帧占比 × 30

# 4. 创新亮点

本项目的算法设计有以下创新点，适合作为高中生科创项目的亮点展示：

## 亮点一：基于 MDF 的动态基线疲劳检测

传统 sEMG 疲劳检测使用校准时的 MDF 作为固定基线，存在日间漂移问题。本算法创新地采用「每次收缩独立基线」策略：

- 当激活度从 <20% 跨越到 >20% 时，自动捕获当前 MDF 作为本次收缩的基线
- 这样即使电极有轻微位移，也能正确反映本次收缩内的疲劳趋势
- 该设计提高了算法的实用性和长期稳定性

> **学术价值**：这是运动生理信号处理的常见改进方向，相关思路在文献 [3] Merletti (2004) 中也有提及。作为高中生项目，自主实现此功能体现了算法设计的问题意识。

## 亮点二：自适应 EMA 平滑参数

MDF 的 EMA 平滑参数 $\alpha$ 不是固定值，而是根据信号状态自适应调整。固定 $\alpha$ 的 EMA 是常见做法，但本算法根据生理含义自适应调整 $\alpha$，这是一个体现「算法针对性设计」的亮点。

## 亮点三：极低资源的嵌入式实现

Arduino UNO R4 WiFi 仅有 32KB RAM 和 48MHz 主频，在这样的限制下实现实时 FFT + MDF 计算是一个工程亮点：

- FFT 旋转因子预计算并存储在 BSS 段（`static` 数组），避免每次计算重新生成
- 使用 256 点 FFT（而非 512 点），在满足频率分辨率的同时控制 RAM 占用
- 环形缓冲区设计避免大数组的栈分配（RA4M1 主栈仅 1KB）
- 固件编译后 RAM 使用率约 **58.7%**（约 19KB / 32KB），在严格限制下成功实现全部功能

## 亮点四：完整的本地 + 云端数据闭环

固件端完成全部算法计算，结果（RMS、MDF、激活度、疲劳度）通过 WiFi 每 3 秒上传至微信云开发平台，用户可在微信小程序上实时查看，并支持历史数据查询。相比传统「传感器采集 → 传输到计算机 → 计算机软件分析」的流程，本设备实现了「采集 → 计算 → 显示」全链条的嵌入式实现。

# 5. 优点与缺点分析

## 5.1 优点

| 优点 | 说明 | 学术/实用价值 |
|------|------|--------------|
| 实时性强 | 固件端 10Hz 计算频率，延迟 < 100ms | 适合运动中的实时反馈 |
| 低功耗便携 | 腕带式设计，USB 供电 | 适合运动科学和康复场景 |
| MDF 动态基线 | 每次收缩自动重新校准基线 | 提高长期使用的稳定性 |
| 自适应 EMA | 根据信号状态调整平滑参数 | 兼顾响应速度和稳定性 |
| 信号质量指示 | 实时评估信号质量 | 提高数据可靠性 |
| 成本极低 | BOM 约 200 元 | 适合教育推广 |
| 云端数据 | 微信小程序 + 云开发 | 用户友好，数据可回溯 |

## 5.2 缺点与局限性

| 缺点 | 原因 | 改进方向 |
|------|------|---------|
| 频率分辨率有限 | 256 点 FFT @ 1000Hz → ~3.9 Hz/bin | 增加 FFT 点数（需更多 RAM） |
| MDF 计算依赖频谱质量 | 电极接触不良时功率谱异常 | 已通过信号质量评估缓解 |
| 校准需要主动配合 | 用户需完成 10s 放松 + 15s 最大收缩 | 可考虑无校准模式 |
| 单通道 sEMG | 仅采集一块肌肉 | 硬件扩展（多路 ADC） |
| 疲劳度需主观验证 | MDF 下降与主观疲劳的相关性 | 下一步：受试者对照实验 |
| UNO R4 WiFi 算力限制 | 无法运行更复杂的频域特征 | 升级到 STM32 或 ESP32 |

> **说明**：作为高中生科创项目，坦诚说明算法的局限性，并在答辩中展示对改进方向的思考，往往比只讲优点更容易获得评委认可。

# 附录：核心公式汇总

| 符号 | 公式 | 说明 |
|------|------|------|
| RMS | $\sqrt{(1/N)\sum (x[i] - DC_{bias})^2}$ | 肌肉激活强度（mV） |
| MDF | $\int P(f)df = 0.5 \times P_{total}$ | 频谱中心频率（Hz） |
| 激活度 A | $\text{clamp}((RMS - RMS_r) / (RMS_a - RMS_r) \times 100, 0, 100)$ | 归一化收缩强度（%） |
| 疲劳度 F | $\text{clamp}((MDF_{bl} - MDF_{cur}) / MDF_{bl} \times 100, 0, 100)$ | 基于 MDF 下降的疲劳指数（%） |
| 疲劳度平滑 | $F(t) = F(t-1) \times 0.9 + F_{raw}(t) \times 0.1$ | EMA 平滑输出 |

*注：$RMS_r$ = 放松状态 RMS，$RMS_a$ = 主动收缩 RMS，$MDF_{bl}$ = 动态基线或 $MDF_{relax}$，$N = 256$。*
